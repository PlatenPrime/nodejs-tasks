Создайте приложение на Node.js, 
которое кэширует данные с ограничением по времени. 
Когда кэш истекает, 
данные должны автоматически обновляться 
из внешнего источника (например, из API) при следующем запросе.

➡️ Пример:

• node app.js fetch — получает данные из кэша, 
а если кэш истек, 
то запрашивает данные заново и обновляет кэш.
